# ImageSharp

The ImageSharp component is dependent on the external package `SixLabors.ImageSharp` which is available on [NuGet](https://www.nuget.org/packages/SixLabors.ImageSharp/1.0.1). Aero CMS currently supports version `1.0.1` of ImageSharp.

## Installation

The ImageSharp processor can be installed by adding the following NuGet package:

~~~ csharp
PM> install-package Aero.ImageSharp
~~~
